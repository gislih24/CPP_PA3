## How to run

Compile:
```bash
make
```

Create a file called `input.txt` and write an equation to be parsed into a tree.

Build tree: 
```bash
./bin/ast_program build tree.txt input.txt
```

Evaluate solution for syntax tree: 
```bash
./bin/ast_program eval tree.txt
```
